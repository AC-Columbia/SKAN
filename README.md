# SKAN

## How to Compile and Run SKAN

Install all requirements using homebrew:

```brew install cmake```

```brew install llvm```

```brew install pkg-config```

```opam install llvm```

```brew install opencv@2```

### SKAN

Compile:

```make```

To compile and run {BASE}.sk:

```./run.sh {BASE].sk```

```./{BASE}.out```
